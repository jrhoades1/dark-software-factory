# Entering Insurance Jobs in ServiceTitan

> Reference: `csr-training` skill
> Step-by-step process for creating insurance jobs in ServiceTitan.

---

## Creating the Job

### 1. Search for Insurance
- Use the internal job under ALD Facility the insurance name

### 2. Create Parent-Child Account
- Click "Parent-Child Account"
- Set up the account hierarchy

### 3. Enter Homeowner's Last Name and Claim Number
- The claim number goes in the account record

### 4. Enter Job Type
- Choose the correct job type for the work being done

### 5. Select Business Type
- Set business type to **"Insurance"**

### 6. Add Tags
- Select **"Insurance"** tag
- Add **"Public Adjuster"** tag if applicable

### 7. Set Marketing Campaign
- Set the Marketing Campaign to the **insurance company requesting service**

### 8. Enter Adjuster Information
- In the summary section, add the **adjuster's name** and **phone number** for the technician to contact

---

## Customer Side for Insurance

### 1. Add Insured's Information
- Enter the insured person's (homeowner's) name and phone number

### 2. Email and Memo for Insurance
- For the email, use: **insurancejobs99@gmail.com**
- In the Memo section, write **"Office Billing"**

### 3. Add Insurance Adjuster as Second Contact
- Enter the insurance adjuster's phone number as the second contact
- If there is a Public Adjuster, add them as an additional contact
